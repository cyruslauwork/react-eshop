# react-eshop

### A React App Without a Framework

- Maximize flexibility in adopting React into existing web applications.

- First time installation steps for the project (npm init has been made):
    ```javascript
    npm i
    ```

- To run the project:
    ```javascript
    npm run start
    ```

- Go to eShop:
    ```bash
    http://127.0.0.1:4000/
    ```
